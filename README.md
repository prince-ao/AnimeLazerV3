## Features

- Recent Anime Release.
- Popular Anime.
- Top Airing Anime.
- Horror Anime.
- Fiction Anime.
- Action Anime.
- Search Anime.
- Add to favorites.
- Sign into My Anime List.
- Update MAL list.
- Episode progress.
- Ad free.

## Technologies used

- Javascript
- React Native
- React Navigation
- Expo
- SQLite
- Express
- Axios
- Puppeteer
- Cheerio
- Jsonwebtokens
